# Web Piano
### A web based virtual piano

## Requirements
- Python
  - For running Flask
- NPM
  - For installing js deps and building

## Installation
```
chmod +x install.sh && ./install.sh
```

## Usage
#### Start with:
```
./web.py
```
#### Then connect to the local ip Flask created
